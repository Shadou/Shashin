## Git
```sh
git init
git add .

git commit -m "submit"

git remote add origin git@github.com:Shadou/Shashin.git
git push -u origin master

git clone https://github.com/Shadou/Shashin.git
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
